# Climate-AICA Project Context

## Overview
AI-powered Climate data literacy engine, being developed as an AICA Award
application. Focus on democratizing climate data access and understanding.

## Modules
1. **Misconception Detector** — AI tool identifying common climate data misunderstandings
3. **Data Narrator** — AI-powered visualization narration system
5. **COP Advisor** — Negotiation data bridge for COP delegates

## Framework
Discover → Understand → Question → Build → Act pipeline

## Target Users
- Climate negotiators
- Researchers (31% face data access barriers)
- SIDS (Small Island Developing States) governments
